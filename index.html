The Basic Language of the Web: HTML

Posted by Laura Jones on Monday, October 13th. 2024

All modern website and web applications are built using three fundadamentaltechnologies: HTML, CSS, and JavaScript. These are the languages of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learnt it.

What is HTML?

HTML stands for HyperText Markup Language. It's a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe different types of content: paragraphs, links, headings, Images, video etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  • The Opening Tag
  • The Closing Tag
  • The Actual Element

You can learn more at MDN Web Docs.
